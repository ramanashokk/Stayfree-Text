# Stayfree

An AI-powered voice-to-text Progressive Web App (PWA).

## Features

- 🎤 Speech Recognition
- 📱 Installable Android PWA
- 🌐 Works from the browser
- ⚡ Fast voice transcription
- 🔒 No app store required
